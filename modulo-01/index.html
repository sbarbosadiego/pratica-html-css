Uso de Imagens e Favicon

É possível utilizar imagens dentro de arquivos HTML, porém é importante verificar se o uso da mesma é liberado, neste repositório há uma explicação sobre o uso de imagens e direitos reservados.
Existem diversos sites que disponibilizam imagens para uso de forma gratuita, e alguns é pago uma espécice de assinatura que permite o uso da image, entretato é sempre bom verificar até em sites gratuitos sobre a permissão do uso da imagem, alguns sites com imagens gratuitas: Pexels, Pixabay, Freepik e Unsplash.

Basicamente a insersão de uma imagem e dimensionamento será através da seguinte tag:
<img src="local do arquivo" alt="texto alternativo" width="300 (tamanho em pixels)">

Em caso de dúvidas em como fazer o uso de imagens com HTML basta clicar no link.

private void salvarProduto() {
    modelProduto.setProdutoNome(this.jtfNomeProduto.getText().toUpperCase());

                JOptionPane.ERROR_MESSAGE);
    }
}
