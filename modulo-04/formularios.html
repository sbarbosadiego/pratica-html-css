Teste Com Formulários

Primeiro Formulário

Por padrão o método utilizado em formulários é o GET, este pode ser usado em que se trabalha com o envio de dados não sensíveis, este mesmo método carrega as informações preenchidas na URL.

Segundo Formulário

Ao trabalhar com dados sensíveis é preciso alterar o método do formulário para POST, caso não seja utilizado os dados serão exibidos na URL. Ainda não é maneira mais segura de fazer o envio de informações, sendo necessário o uso de procotolo HTTPS.
